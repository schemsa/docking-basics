# 004 - Calculating the ligand center

Today I calculated the approximate geometric center of the ligand `E20`.

## Input file

- `prep/ligand_E20.pdb`

## Goal

The goal was to estimate the center of the ligand binding site.

This center can later be used as the center of the docking search box in AutoDock Vina.

## Command used

```bash
awk '/^HETATM/ {x+=$7; y+=$8; z+=$9; n++} END {print "center_x =", x/n; print "center_y =", y/n; print "center_z =", z/n}' prep/ligand_E20.pdb
