002 - Identifying ligands in a PDB file

I learned how to inspect a `.pdb` file from the terminal.

## Commands used

```bash
grep HETATM data/1EVE.pdb | head

This command shows the first non-standard atoms in the PDB file.

grep HETATM data/1EVE.pdb | awk '{print $4}' | sort | uniq

This command lists unique residue names from HETATM records.

Key concepts
ATOM records usually correspond to standard protein atoms.
HETATM records usually correspond to non-protein molecules.
HOH means water.
Other residue codes may correspond to ligands, ions, or cofactors.
