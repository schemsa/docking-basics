# 003 - Extracting the ligand and receptor from a PDB file

Extracted the main ligand and the receptor from the original protein structure file.

## Input file

* `data/1EVE.pdb`

The original PDB file contains several types of records, including:

* protein atoms
* ligand atoms
* water molecules
* sugars or other non-standard residues
* metadata

For molecular docking, the receptor and ligand are usually prepared as separate files.

## Files created

* `prepared/ligand_E20.pdb`
* `prepared/receptor_1EVE.pdb`

---

## 1. Extracting the ligand

The ligand selected from the PDB file was `E20`.

### Command used

```bash
grep "^HETATM" data/1EVE.pdb | grep "E20" > prepared/ligand_E20.pdb
```

### Explanation

The command works in three parts:

```bash
grep "^HETATM" data/1EVE.pdb
```

This selects only `HETATM` records from the original PDB file.

```bash
grep "E20"
```

This filters only the lines corresponding to the ligand `E20`.

```bash
> prepared/ligand_E20.pdb
```

This saves the result into a new file.

### Check commands

```bash
ls prepared
head prepared/ligand_E20.pdb
wc -l prepared/ligand_E20.pdb
```

### Notes

* `HETATM` records usually correspond to non-standard molecules.
* `E20` is likely the main ligand or inhibitor in this structure.
* `HOH` corresponds to water molecules.
* `NAG` corresponds to N-acetylglucosamine, a sugar often associated with glycosylation.

---

## 2. Extracting the receptor

The receptor was extracted by keeping only the `ATOM` records from the original PDB file.

### Command used

```bash
grep "^ATOM" data/1EVE.pdb > prepared/receptor_1EVE.pdb
```

### Explanation

```bash
grep "^ATOM" data/1EVE.pdb
```

This selects only the standard protein atom records.

```bash
> prepared/receptor_1EVE.pdb
```

This saves the protein-only structure into a new file.

### Check commands

```bash
ls prepared
head prepared/receptor_1EVE.pdb
wc -l prepared/receptor_1EVE.pdb
```

### Notes

This is a simplified receptor preparation step.

Later, receptor preparation may also include:

* removing water molecules
* adding hydrogens
* assigning charges
* converting files to `.pdbqt`
* defining the docking search box

---

## Summary

From the original file:

```text
data/1EVE.pdb
```

I created two separate files:

```text
prepared/ligand_E20.pdb
prepared/receptor_1EVE.pdb
```

This separation is necessary because molecular docking requires:

```text
receptor + ligand
```

instead of one combined structure file.
