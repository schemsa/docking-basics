# 001 - Protein structures

Learning about protein structure files.

## Key concepts

- A protein structure can be stored in a `.pdb` file.
- PDB stands for Protein Data Bank.
- A `.pdb` file contains atom coordinates.
- Molecular docking needs a receptor, usually a protein.
- The ligand is the small molecule that will be docked into the receptor.

Protein: Acetylcholinesterase
